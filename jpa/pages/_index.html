Gemini JPA

Modular JPA Support

The Gemini JPA project is about modular implementations of Java Persistence API technology. This project provides a JPA provider that follows the OSGi JPA specification approach and that clients can use to get JPA support in OSGi.

The modules run on the Equinox OSGi framework and leverage the OSGi bundle model of packaging and life cycle activation. They integrate and expose the EclipseLink JPA provider as OSGi services.

Modularity

The Gemini JPA project is packaged as a small number of separate and standalone module.

Independence

The Gemini JPA subproject is downloadable and executable independently from the others Gemini subprojects.

Free and Easy

Like everything else at Eclipse, getting started is as easy as downloading and trying it out!
